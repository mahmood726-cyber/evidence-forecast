Evidence Forecast — Phase 1

Probabilistic forecast of meta-analysis conclusion flips. Each card is a cryptographically signed JSON + HTML bundle containing: current pooled effect, P(flip within 24 months), burden-weighted representativeness, and TruthCert HMAC-SHA256 signature.

DEV MODE. Cards below are generated from stub effects anchored to published primary MAs (Vaduganathan 2022, Packer 2025 SUMMIT, Zinman 2015 EMPA-REG) and a synthetic-data flip-forecaster. Real integration (CardioSynth adapter match + MetaAudit-trained model + registry-first population weights) is the user-present release step. See README for the six steps to reach real Phase-1 release.
